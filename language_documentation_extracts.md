`.E` - Contains math constant e.

`.cos(<T> value)` => `<T>`  
`.tan(<T> value)` => `<T>`  
`.pow(<T> base, <T> power)` => `<T>`  
`.max(<T> value1, <T> value2)` => `<T>`  
`.min(<T> value1, <T> value2)` => `<T>`  
`.abs(<T> value)` => `<T>`  
`.sqrt(<T> value)` => `<T>`
